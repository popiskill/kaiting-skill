# 开庭 Skill - 使用说明

## 快速开始

```
/开庭
```

然后提交你要审判的 Skill。

## 命令列表

| 命令 | 说明 |
|------|------|
| `/开庭` | 开始审判流程 |
| `/开庭-列表` | 查看已审判的 Skill 档案 |
| `/开庭-对比 {名字1} {名字2}` | 对比两个 Skill |

## 审判流程

1. **提交被告** — 提供Skill路径或链接
2. **法庭调查** — 分析 Skill 结构和质量
3. **质询环节** — 尖锐提问，暴露问题
4. **最终宣判** — 评分 + 等级 + 排名

## 评分维度

| 维度 | 满分 | 说明 |
|------|------|------|
| 实用性 | 25 | 真解决问题？使用成本合理？|
| 文档质量 | 20 | 清晰？示例完整？|
| 代码质量 | 20 | prompt 专业？工具可用？|
| 创新性 | 15 | 独创？换皮？|
| 完整性 | 20 | 功能完整？边界处理？|

## 等级说明

| 等级 | 分数 | 含义 |
|------|------|------|
| 🔥 夯得一批 | 90-100 | 牛逼，直接装 |
| 👍 相当可以 | 75-89 | 值得一用 |
| 😐 也就那样 | 60-74 | 能用，别期望太高 |
| 🤔 有点拉胯 | 40-59 | 问题不少 |
| 💩 拉得一批 | 0-39 | 浪费人生 |
